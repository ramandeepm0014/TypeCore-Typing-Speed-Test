# ⌨️ TypeCore - Typing Speed Test Application

TypeCore is a Streamlit-based typing speed test application that measures WPM, accuracy, and typing performance. It includes multiple difficulty levels, real-time feedback, a performance dashboard with analytics, test history tracking, and a competitive leaderboard system.

## 🚀 Features

* Real-time typing speed (WPM) calculation
* Accuracy tracking
* Multiple difficulty levels (Easy, Medium, Hard)
* Instant typing feedback
* User performance dashboard
* Progress tracking with charts
* Leaderboard ranking system
* Test history storage

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Altair

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
